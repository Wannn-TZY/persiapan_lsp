Tutorial Setup Project Laravel "persiapan_lsp" di Perangkat Baru

1. Persyaratan
- PHP 8.1 atau lebih tinggi
- Composer
- Node.js + npm
- Database: MySQL / MariaDB / PostgreSQL
- Git

2. Clone repository
1) Buka terminal di folder tujuan
2) Jalankan:
   git clone 'https://github.com/Wannn-TZY/persiapan_lsp'
   cd persiapan_lsp

3. Install dependency PHP
   composer install

4. Install dependency frontend
   npm install

5. Siapkan file environment
   copy .env.example .env
   (atau di Git Bash: cp .env.example .env)

6. Ubah file .env
Sesuaikan konfigurasi database:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=persiapan_lsp
DB_USERNAME=root
DB_PASSWORD=

7. Buat database
Buka phpMyAdmin atau terminal MySQL, lalu buat:
   CREATE DATABASE persiapan_lsp;

8. Generate APP_KEY
   php artisan key:generate

9. Jalankan migration + seeder
   php artisan migrate --seed

10. Build frontend (opsional)
- Development:
   npm run dev
- Build produksi:
   npm run build

11. Jalankan server Laravel
   php artisan serve

12. Buka aplikasi
   http://127.0.0.1:8000

13. Struktur login
- Login siswa: NIS + password
- Login walas: NIG + password

14. Troubleshooting umum
- Jika `composer install` gagal: pastikan Composer terpasang.
- Jika `php artisan` tidak ditemukan: pastikan terminal berada di root project.
- Jika database error: periksa .env dan akses database.
- Jika `vendor/` tidak ada: jalankan ulang `composer install`.

15. Tambahan penting
- Pastikan `.env` bukan file yang dikirim ke repo publik.
- Jika pakai XAMPP/Laragon: gunakan PHP path yang benar.
- Jika ingin reset database dan seed ulang:
   php artisan migrate:fresh --seed

Catatan:
Project ini menggunakan Laravel 10, Vite, dan login sederhana berbasis session.
Jika ada error login siswa walau data seeder sudah ada, periksa:
- kredensial NIS/NIG dan password
- apakah tabel `nilais` sudah berisi data untuk siswa
- route dashboard / nilai show sudah ada.
